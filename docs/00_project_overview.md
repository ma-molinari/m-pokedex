# 00 - Visão Geral do Projeto

Este documento fornece uma visão geral de alto nível da aplicação Pokedex.

## Objetivo

A aplicação é uma Pokedex digital que permite aos usuários visualizar informações sobre diferentes Pokémon. Os usuários podem navegar sequencialmente pelos Pokémon, vendo detalhes como tipo, estatísticas e uma breve descrição.

## Funcionalidades Principais

- **Visualização de Pokémon**: Exibe um cartão de Pokedex para o Pokémon atual.
- **Navegação**: Permite que o usuário navegue para o próximo Pokémon ou para o anterior usando botões ou as teclas de seta do teclado.
- **Carregamento Dinâmico**: Os dados do Pokémon são carregados de forma assíncrona da PokeAPI.
- **Estilização Dinâmica**: A cor de fundo da aplicação muda para combinar com o tipo principal do Pokémon sendo exibido.
- **Pré-carregamento de Imagens**: As imagens do Pokémon seguinte e anterior são pré-carregadas para uma navegação mais fluida.
